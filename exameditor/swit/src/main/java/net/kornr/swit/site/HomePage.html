Welcome to Swit

The Swit project aims at providing the finest sugar, not to say the switest, to java with a specific focus on Wicket. There's three parts in Swit:
  • A button generator
  • A border generator
  • Layout managers

Layout Managers for Wicket

Well, not really, but if you're tired to create 2 or 3 columns CSS layouts, or even worse, to recursively embed n-columns layout, you know it's good to have a way to do it automatically.
The details of the CSS technique at http://matthewjamestaylor.com/blog/perfect-multi-column-liquid-layouts
This is a 3-column layout with fixed 100-pixel columns on both sides. The center streches out with the page size (try resizing this window).
The good thing with having an automatic generator, is that you can recursively embed layouts in layouts, it takes care of generating css elements with different, unique names.
This is a 2-column layout, specified with a percentage unit. The left column is 33% of the width, the remaining is for the central column.
Just so that you know, there's not a single image on this page that has not been automatically generated.
